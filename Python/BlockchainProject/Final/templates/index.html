Blockchain Project
Click on an above navigation link to view a relevant section of the site.

Web interface for working with Blockchain infrastructure implemented though Python Flask server.

The Blockchain infrastructure at work here is semi-decentralized, only authorized personnel can add records to the chain, but the entire blockchain itself is publicly accessible.

The records contained within this Blockchain relate to vehicle mileage information, this provides a simplistic structure to the overall chain which is easy to use and comprehend.

The Blockchain page is publicly accessible, while the Add Block section is reserved for authenticated personnel only.
